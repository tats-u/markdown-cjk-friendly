# Security Policy

## Supported Versions

Supports the latest major versions. Backports to versions that are not the latest major version will be considered based on npm download frequency, but there is no guarantee that they will be implemented.

## Reporting a Vulnerability

Please ensure to initiate your report by pressing the "Report a Vulnerability" button located at the top right of this page. This does not apply to requests for updates to transitive dependencies already listed in the GitHub Security Advisory.
